# Premium FAANG-Level Portfolio Website

A high-performance, visually stunning personal portfolio website built for software engineering applications. Designed with a sophisticated dark, warm-toned palette and smooth, physics-based interactions.

## 🚀 Tech Stack

- **Framework:** [Next.js 14 (App Router)](https://nextjs.org/)
- **Animations:** [GSAP 3](https://gsap.com/) (ScrollTrigger, Timelines)
- **Smooth Scroll:** [Lenis](https://lenis.darkroom.engineering/)
- **Styling:** [Tailwind CSS](https://tailwindcss.com/)
- **Components:** [shadcn/ui](https://ui.shadcn.com/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Forms:** React Hook Form + Zod
- **Type Safety:** TypeScript

## ✨ Key Features

- **Smooth Scrolling:** Custom physics-based scroll experience using Lenis, synchronized with GSAP's animation ticker.
- **Kinetic Typography:** Custom character-splitting utility for staggered text entrance animations.
- **Dynamic Hero:** Full-viewport hero section with parallax effects and animated noise texture.
- **Interactive Gallery:** Masonry-style layout with a custom GSAP image lightbox.
- **Project Showcase:** Featured and grid-based project cards with hover-scaling and stagger effects.
- **Technical Arsenal:** Categorized skill grid with proficiency levels and animated tickers.
- **Responsive Form:** Fully validated contact form with real-time feedback and mock API integration.

## 🛠️ Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 📦 Production

To create an optimized production build:

```bash
npm run build
```

To preview the production build:

```bash
npm run start
```

## 🌐 Deployment

The easiest way to deploy this project is via [Vercel](https://vercel.com/new).

1. Push your code to GitHub.
2. Connect your repository to Vercel.
3. Configure your environment variables if needed.
4. Deploy!
